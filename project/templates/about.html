Project Overview

RetailAI is a web-based platform built to address the critical challenge of inventory management in local retail markets. By applying supervised machine learning algorithms — specifically Linear Regression and Random Forest — the system analyzes historical sales data and generates accurate demand forecasts.

The system reduces overstocking losses and prevents stockouts by providing actionable inventory recommendations backed by data science.

AI Prediction Models

📈 Linear Regression

Models the relationship between time/features and demand as a linear equation: Demand = β₀ + β₁X₁ + β₂X₂ + ε. Fast, interpretable, works well for datasets with linear trends. Best for stable products.

🌳 Random Forest

An ensemble of 100 decision trees that vote on predictions. Handles non-linear patterns, outliers, and feature interactions. Provides feature importance scores. Best for complex retail data.
